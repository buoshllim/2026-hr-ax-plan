01. Context

패러다임의 전환: 도구에서 주체로

The Tool Era (PPT/Excel)

사람이 일의 주체이며, 도구는 사람의 보조 수단으로 활용되었습니다. 숙련도 차이가 개인의 생산성 격차를 만들었으나, 일의 구조는 변하지 않았습니다.

The Agent Era (AI)

업무 자체를 에이전트에 위임합니다. 이제 사람은 Workflow를 설계하는 'Orchestrator'가 되며, 다수의 에이전트 운용을 통해 개인이 조직 수준에 버금가는 성과를 창출하게 됩니다.

"지금까지 파워포인트와 엑셀 활용 역량이 기본이었던 것처럼,
조만간 에이전트를 직접 설계하고 시스템과 연동하여 활용하는 능력이 구성원의 기본값이 될 것입니다."

02. Vision

우리가 가야 할 곳

전 구성원의 Orchestrator 化

모든 구성원이 다수의 에이전트를 조율하며, 무엇을(What)・왜(Why) 해야 하는지에 집중합니다. 증원 보다는 에이전트 확장을 통해, 구성원 능력을 증강(Augment)시켜 성과를 극대화합니다.

  • 실행 주체 AI Agents (업무 자동화 및 초안 작성)
  • 구성원 역할 가치 판단, 룰 설계 및 최종 의사결정
  • 확장 방식 에이전트 복제 및 병렬 처리

04. Approach

2-Layer 추진 전략

Layer 1: HR Self-Build

"HR이 직접 만들고
먼저 경험한다"

Profiling, Career, Succession Agent를 순차적으로 구축하여 HR 스스로가 첫 번째 Pilot이 됩니다.

Layer 2: Lead Evolution

"성공 경험을 전사로
확산하고 주도한다"

HR의 Pilot 모델을 기반으로 타 조직의 도메인 Agent 전환을 리드하여 SK스퀘어 전체의 Agent Native를 완성합니다.
